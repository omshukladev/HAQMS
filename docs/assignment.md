# **Figital Labs Full Stack Web Development Internship Assignment**

## **Overview**

Welcome to the HAQMS Engineering Evaluation Assignment.

You will be working on an existing full-stack hospital management application that intentionally contains:

- security vulnerabilities
- performance bottlenecks
- database inefficiencies
- frontend issues
- concurrency problems
- incomplete features

Your objective is to identify, debug, optimize, secure, and improve the application as much as possible.

# **Repository**

GitHub Repository:

<https://github.com/TechFigitablLabs/HAQMS.git>

Please fork the repository before starting your work.

# **Tech Stack**

- Frontend: Next.js + Tailwind CSS
- Backend: Node.js + Express
- Database: PostgreSQL + Prisma ORM

# **Setup Instructions**

Please follow the setup instructions mentioned in the repository README carefully.

You are expected to:

- Setup the project locally
- Run the database and seed data
- Explore the application
- Identify issues
- Implement fixes and improvements

# **Important Notes**

- You are allowed to use AI tools such as ChatGPT, Claude, Cursor, GitHub Copilot, etc.
- However, your understanding, implementation quality, and engineering decisions will be evaluated carefully.
- Focus on writing maintainable, scalable, and production-quality code.

You are NOT expected to fix everything.

We are more interested in:

- prioritization
- debugging approach
- code understanding
- reasoning behind your fixes
